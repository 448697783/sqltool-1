# sqltool
# 功能
1、可替换现有数据库客户端工具  
2、支持dml、ddl  
3、在同时管理多个类型数据库同时，避免切换各种客户端  
4、支持表名、字段名自动补全  
5、支持使用别名自动补全  
6、支持sql格式化  
7、支持查看ddl语句  
8、支持表格编辑  
9、支持关联pdm中描述，减少查数据边翻看pdm麻烦，特别对新接手项目的开发人员  
10、支持跨数据库迁移数据（单表迁移到多表、多表迁移到单表），可在项目升级数据库时使用  
11、支持迁移时脚本保存  
12、支持生成随机数据  

# 支持数据库类型
1、oracle  
2、db2  
3、mysql  
4、sqlserver  
5、redis
